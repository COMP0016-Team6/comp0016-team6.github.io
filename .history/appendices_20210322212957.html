Deployment Manual

Running the App Locally

Prerequisites:

  • git
  • npm or yam

General Installation

  • git clone https://github.com/COMP0016-Team6/NHS-Patient-Dashboard.git
  • npm install react

To Run App

  1. Connect to the database
    1. Open the SQL Shell and login to database
    2. Open db.js in server directory
    3. Revise line 5 and 6 to your PostgreSQL username and password
      user: "",
      password: "",
  2. Open server directory
    cd server
  3. Deploy the server
    npm run dev
  4. Open a new terminal
  5. Open client directory
    cd client
  6. Run the app
    npm start
